This currency converter is developed with the Fixer Currency API. As on date (August 5th 2017), it supports 32 countries and the rates are published by the European Central Bank and updated at 4PM Central European Time (CET).

This application is developed using Flask web microframework and Python and uses a sqlite3 database. This web app is hosted on a Raspberry pi Model B.
